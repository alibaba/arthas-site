Java 应用诊断利器

Arthas 是Alibaba开源的Java诊断工具，深受开发者喜爱。

特性一览

  • Dashboard

    实时查看系统的运行状况

  • 查看入参/返回值/异常

    查看函数调用的参数，返回值和异常

  • 在线热更新

    jad/sc/redefine 一条龙热更新代码

  • 类冲突

    秒解类冲突问题，定位类加载路径

  • 性能热点

    快速定位应用的热点，生成火焰图

  • WebConsole

    在线诊断，点开网页诊断线上应用
